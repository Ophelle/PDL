Comparison of programming languages (string functions) - Find character

 
 
Formulaire




Returns the position of the start of the first occurrence of the character char in string. If the character is not found most of these routines return an invalid index value – -1 where indexes are 0-based
0 where they are 1-based – or some value to be interpreted as Boolean FALSE. This can be accomplished as a special case of #Find
with a string of one character; but it may be simpler or more efficient in many languages to locate just one character. Also
in many languages
characters and strings are different types
so it is convenient to have such a function.
